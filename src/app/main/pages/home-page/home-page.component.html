Bienvenido a nuestra plataforma web, donde te brindamos la oportunidad de realizar evaluaciones que son fundamentales para el desarrollo académico. Aquí, puedes acceder al "Test de Eficacia Lectora (TECLE)," el "Test de Eficacia Ortográfica (TEO)," y el "Test de Eficacia del Cálculo Aritmético (TECA)," diseñados específicamente para estudiantes de primaria.

Este sitio web fue desarrollado por estudiantes del segundo semestre de la carrera de Desarrollador de Software en la Universidad Católica del Uruguay. Nuestra meta es automatizar estas pruebas, asegurando que el tiempo destinado a cada evaluación se cumpla eficientemente. Esta automatización no solo agiliza el proceso, sino que también garantiza que los resultados se almacenen fácilmente en una base de datos de acceso sencillo. De esta manera, se facilita la evaluación del nivel de la clase en la que se aplicaron estas pruebas. Únete a nosotros y explora una nueva era de evaluaciones educativas eficientes y precisas directamente desde nuestra página web. ¡Te invitamos a descubrir el futuro de la evaluación educativa!
